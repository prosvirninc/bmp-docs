# Эффективные TRP

## Входные данные

| Источник | Описание |
|----------|----------|
| Файл охватов кампании | Рекламодатель, Кампания, TRP кампании, Средний охват, Нижняя граница частоты, Верхняя граница частоты, Охват 1+, Охват 2+, … |
| Пул открытых охватов | Аналогичная структура по историческим кампаниям |

---

## Расчет индекса эффективных контактов

### Пул кампаний

Пул фильтруется по трем условиям:
- Исключаются кампании того же рекламодателя.
- TRP кампании из пула попадает в диапазон ±10% от TRP анализируемой кампании. Если кампаний с таким допуском меньше 10, допуск расширяется с шагом 5% до максимума ±25%.
- Средний охват попадает в диапазон `[Средний охват × 0.5; Средний охват × 1.5]`.

### Доля эффективных контактов

Границы эффективной частоты берутся из файла кампании (поля `'Нижняя граница частоты'` и `'Верхняя граница частоты'`). Для краткости обозначим их `'Мин. частота'` и `'Макс. частота'`. Для n от 'Мин. частоты' до 'Макс. частоты':

```
'TRP эфф.' = Σ ('Охват n+' − 'Охват (n+1)+') × n − 1 + 'Охват Макс. частота+' × 'Макс. частота'

'Доля эфф. контактов' = 'TRP эфф.' / 'TRP камп.'
```

### Рыночное значение

По кампаниям пула с теми же границами частоты:

```
'Доля эфф. контактов рын.' = среднее('Доля эфф. контактов')
```

### Индекс

```
'Оценка эфф. контактов' = ('Доля эфф. контактов камп.' / 'Доля эфф. контактов рын.') × 100
```

Значение `> 100` означает, что кампания эффективнее среднерыночной.

---

## Расчет закрытых охватов

Каждой частоте `n` сопоставляется закрытый охват `'Охват n+' − 'Охват (n+1)+'` и его TRP-вклад.

Максимальная частота расчета `'N max'` определяется по TRP кампании:

| TRP | 'N max' |
|-----|-------|
| < 1200 | 25 |
| 1200–1700 | 40 |
| 1700–2200 | 50 |
| ≥ 2200 | 60 |

### Разбивка по зонам частот

**До `'Мин. частоты'` — неэффективные контакты:**

```
'TRP эфф.' = 0
'TRP неэфф.' = ('Охват n+' − 'Охват (n+1)+') × n
```

**От `'Мин. частоты'` до `'Макс. частоты'` — эффективные контакты:**

```
'TRP эфф.' = ('Охват n+' − 'Охват (n+1)+') × n
'TRP неэфф.' = 0
```

**От `'Макс. частоты' + 1` до `'N max' − 1` — смешанная зона:**

```
'Закрытый охват' = 'Охват n+' − 'Охват (n+1)+'
'TRP эфф.' = 'Закрытый охват' × 'Макс. частота'
'TRP неэфф.' = 'Закрытый охват' × (n − 'Макс. частота')
```

**Строка `'N max'+` — остаток TRP:**

```
'TRP остаток' = 'TRP камп.' − (сумма TRP уже учтенных строк)
'TRP эфф.' = 'Охват N max+' × 'Макс. частота'
'TRP неэфф.' = 'TRP остаток' − 'TRP эфф.'
```

## Итоговые значения

**Лист «Индекс»:**

| Поле | Описание |
|------|----------|
| Рекламодатель | Рекламодатель кампании |
| Кампания | Название кампании |
| TRP камп. | Суммарный TRP |
| Доля эфф. контактов камп. | TRP_eff / TRP камп. |
| Доля эфф. контактов рын. | Среднее по пулу |
| Оценка эфф. контактов | (Доля камп. / Доля рын.) × 100 |
| Границы эфф. частоты | Строка вида «Мин. частота–Макс. частота» |
| Кол-во камп. в пуле | Размер выборки |

**Лист «Охваты»** — таблица с разбивкой TRP по частотам на эффективные и неэффективные контакты.

## Обновление пула

После расчета каждая кампания добавляется в пул (если ее там еще нет), чтобы будущие расчеты могли ее использовать как референс.
