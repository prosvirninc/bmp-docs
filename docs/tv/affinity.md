# Affinity

## Входные данные

| Источник | Описание |
|----------|----------|
| Palomars | Основная выгрузка |
| Mediascope | GRP и TRP рынка по телеканалам за период кампании |
| Пул Affinity | Пул кампаний с рассчитанными индексами Affinity |

**Используемые столбцы:** Big TVR {ЦА}, Big Sales TVR, Телеканал, Дата, Категория телеканала

---

## Affinity кампании

По каждому телеканалу из выгрузки:

```
'TRP камп.' = Σ Big TVR {ЦА}   — сумма рейтингов роликов на целевую аудиторию
'GRP камп.' = Σ Big Sales TVR  — сумма рейтингов роликов на баинговую аудиторию
```

Суммарно по кампании:

```
'Affinity камп.' = 'TRP камп.' / 'GRP камп.'
```

---

## Affinity по рынку

Запрос к Mediascope за период кампании (сетевое вещание, Big TV, реальные выходы, SPOT+ORBITAL):

```
'Affinity рын.' = 'TRP рын.' / 'GRP рын.'
```

Результаты группируются по телеканалу (орбитальный суффикс " - 0" убирается).

---

## Индекс Affinity

```
'Индекс Affinity' = ('Affinity камп.' / 'Affinity рын.') × 100
```

---

## Пул кампаний

Пул фильтруется по двум условиям:
- Исключаются кампании того же рекламодателя.
- TRP кампании из пула попадает в диапазон `[TRP × 0.5; TRP × 1.5]`.

```
'Средний индекс в пуле' = среднее('Индекс Affinity') по отфильтрованным кампаниям пула
```

---

## Индекс Affinity сплита

```
'Индекс Affinity сплита' = ('Индекс Affinity' / 'Средний индекс в пуле') × 100
```

Значение `> 100` означает, что сплит кампании лучше рыночного среднего.

---

## Распределение по категориям

По категориям телеканалов:

```
'Доля GRP' = 'GRP камп. по категории' / 'GRP камп. итого'
'Доля TRP' = 'TRP камп. по категории' / 'TRP камп. итого'
```

---

## Affinity по телеканалам

Для каждого телеканала рынка:

```
'Affinity камп.' = ('TRP камп. по каналу' / 'GRP камп. по каналу') × 100
'Affinity рын.' = ('TRP рын. по каналу' / 'GRP рын. по каналу') × 100
'Доля TRP' = 'TRP камп. по каналу' / 'TRP камп. итого'
```

Если по каналу нет данных кампании — значения заменяются нулями.

---

## Итоговые значения (лист «Оценка»)

| Поле | Описание |
|------|----------|
| TRP камп. | Суммарный TRP по ЦА |
| GRP камп. | Суммарный GRP по баинговой аудитории |
| Affinity камп. | TRP / GRP камп. |
| Affinity рын. | TRP / GRP рын. |
| Индекс Affinity | (Affinity камп. / Affinity рын.) × 100 |
| Средний индекс в пуле | Среднее по пулу |
| Индекс Affinity сплита | (Индекс Affinity / Средний индекс в пуле) × 100 |
| Кол-во камп. в пуле | Размер выборки |

---

## Обновление пула

После расчета каждая кампания добавляется в пул (если ее там еще нет), чтобы будущие расчеты могли ее использовать как референс.
