# Двойные выходы

Два и более ролика в одном рекламном блоке приводят к пересечению аудитории и накоплению «частотных хвостов». Постановка неэффективна с точки зрения накопления охвата (чем выше % двойных постановок — тем ниже эффективность).

## Входные данные

| Источник | Описание |
|----------|----------|
| Mediascope | Выгрузка по кампании: Big TVR {ЦА}, Телеканал, Телеканал №, Блок ID выхода, Дата, Время начала (0-24) |

> Тематические и орбитальные каналы исключаются перед расчетом.

---

## Определение двойного выхода

Двойной выход — ролик того же рекламодателя, вышедший в одном рекламном блоке не первым по хронологии.

Ролики внутри каждого `'Блок ID выхода'` нумеруются по возрастанию `'Время начала (0-24)'`. Ролик с номером 1 — основной выход, все остальные — двойные.

---

## Расчет TRP

По каждому телеканалу:

```
'TRP'        = Σ Big TVR {ЦА}  (все ролики)
'Double TRP' = Σ Big TVR {ЦА}  (только двойные выходы)
```

---

## Доля двойных выходов

```
'Double share' = 'Double TRP' / 'TRP'
```

При `TRP = 0` значение принимается равным 0.

---

## Итоговые значения

По каждому телеканалу:

| Поле | Описание |
|------|----------|
| Телеканал | Название канала |
| Номер | Порядковый номер канала (для сортировки) |
| TRP | Суммарный TRP кампании |
| Double TRP | TRP двойных выходов |
| Double share | Доля двойных выходов (Double TRP / TRP) |

В конец таблицы добавляется строка «Итого» с агрегированными значениями.
