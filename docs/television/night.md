# Ночные выходы

## Входные данные

| Источник | Описание |
|----------|----------|
| Mediascope | Выгрузка по кампании: Big TVR {ЦА}, Телеканал, Телеканал №, День / Ночь |

> Тематические и орбитальные каналы исключаются перед расчетом.

---

## Определение ночного выхода

Ночным считается ролик, у которого значение столбца `'День / Ночь'` равно `'Ночь'`.

---

## Расчет TRP

По каждому телеканалу:

```
'TRP'       = Σ Big TVR {ЦА}  (все ролики)
'Night TRP' = Σ Big TVR {ЦА}  (только ролики, где 'День / Ночь' = 'Ночь')
```

---

## Доля ночных выходов

```
'Night share' = 'Night TRP' / 'TRP'
```

При `TRP = 0` значение принимается равным 0.

---

## Итоговые значения

По каждому телеканалу:

| Поле | Описание |
|------|----------|
| Телеканал | Название канала |
| Номер | Порядковый номер канала (для сортировки) |
| TRP | Суммарный TRP кампании |
| Night TRP | TRP ночных выходов |
| Night share | Доля ночных выходов (Night TRP / TRP) |

В конец таблицы добавляется строка «Итого» с агрегированными значениями.
