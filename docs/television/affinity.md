# Affinity

## Входные данные

| Источник | Описание |
|----------|----------|
| Mediascope | Выгрузка по кампании: Big TVR {ЦА}, Big Sales TVR, Телеканал, Дата, Категория телеканала |
| Mediascope | GRP и TRP рынка по телеканалам за период кампании |
| База данных | Пул кампаний с рассчитанными индексами Affinity |

---

## Affinity кампании

По каждому телеканалу из выгрузки:

```
'TRP кампании' = Σ Big TVR {ЦА}   — сумма рейтингов роликов на целевую аудиторию
'GRP кампании' = Σ Big Sales TVR  — сумма рейтингов роликов на баинговую аудиторию
```

Суммарно по кампании:

```
'Affinity кампании' = 'TRP кампании' / 'GRP кампании'
```

---

## Affinity по рынку

Запрос к Mediascope за период кампании (сетевое вещание, Big TV, реальные выходы, SPOT+ORBITAL):

```
'Affinity рынка' = 'TRP рынка' / 'GRP рынка'
```

Результаты группируются по телеканалу (орбитальный суффикс " - 0" убирается).

---

## Индекс Affinity

```
'Индекс Affinity' = ('Affinity кампании' / 'Affinity рынка') × 100
```

---

## Пул кампаний

Пул фильтруется по двум условиям:
- Исключаются кампании того же рекламодателя.
- TRP кампании из пула попадает в диапазон `[TRP × 0.5; TRP × 1.5]`.

```
'Средний индекс в пуле' = среднее('Индекс Affinity') по отфильтрованным кампаниям пула
```

---

## Индекс Affinity сплита

```
'Индекс Affinity сплита' = ('Индекс Affinity' / 'Средний индекс в пуле') × 100
```

Значение `> 100` означает, что сплит кампании лучше рыночного среднего.

---

## Распределение по категориям

По категориям телеканалов:

```
'Доля GRP' = 'GRP кампании по категории' / 'GRP кампании итого'
'Доля TRP' = 'TRP кампании по категории' / 'TRP кампании итого'
```

---

## Affinity по телеканалам

Для каждого телеканала рынка:

```
'Affinity кампании' = ('TRP кампании по каналу' / 'GRP кампании по каналу') × 100
'Affinity рынка'    = ('TRP рынка по каналу'    / 'GRP рынка по каналу')    × 100
'Доля TRP'          = 'TRP кампании по каналу'  / 'TRP кампании итого'
```

Если по каналу нет данных кампании — значения заменяются нулями.

---

## Итоговые значения (лист «Оценка»)

| Поле | Описание |
|------|----------|
| TRP кампании | Суммарный TRP по ЦА |
| GRP кампании | Суммарный GRP по баинговой аудитории |
| Affinity кампании | TRP / GRP кампании |
| Affinity рынка | TRP / GRP рынка |
| Индекс Affinity | (Affinity кампании / Affinity рынка) × 100 |
| Средний индекс в пуле | Среднее по пулу |
| Индекс Affinity сплита | (Индекс Affinity / Средний индекс в пуле) × 100 |
| Количество кампаний в пуле | Размер выборки |

---

## Обновление пула

После расчета каждая кампания добавляется в пул (если ее там еще нет), чтобы будущие расчеты могли ее использовать как референс.
