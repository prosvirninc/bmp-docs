# Эффективные TRP

## Входные данные

| Источник | Описание |
|----------|----------|
| Файл охватов кампании | Рекламодатель, Кампания, TRP кампании, Средний охват, Нижняя граница частоты, Верхняя граница частоты, Охват 1+, Охват 2+, … |
| Пул открытых охватов | Аналогичная структура по историческим кампаниям |

---

## Расчет индекса эффективных контактов

### Пул кампаний

Пул фильтруется по трём условиям:
- Исключаются кампании того же рекламодателя.
- TRP кампании из пула попадает в диапазон ±10% от TRP анализируемой кампании. Если кампаний с таким допуском меньше 10, допуск расширяется с шагом 5% до максимума ±25%.
- Средний охват попадает в диапазон `[Средний охват × 0.5; Средний охват × 1.5]`.

### Доля эффективных контактов

Границы эффективной частоты берутся из файла кампании (поля `'Нижняя граница частоты'` и `'Верхняя граница частоты'`). Для краткости обозначим их `'Мин. частота'` и `'Макс. частота'`.

```
'TRP эфф.' = Σ ('Охват n+' − 'Охват (n+1)+') × n    для n от 'Мин. частоты' до 'Макс. частоты' − 1
           + 'Охват Макс. частота+' × 'Макс. частота'

'Доля эфф. контактов' = 'TRP эфф.' / 'TRP кампании'
```

### Рыночное значение

```
'Доля эфф. контактов рынка' = среднее('Доля эфф. контактов')
                               по кампаниям пула с теми же границами частоты
```

### Индекс

```
'Оценка эффективных контактов' = ('Доля эфф. контактов кампании' / 'Доля эфф. контактов рынка') × 100
```

Значение `> 100` означает, что кампания эффективнее среднерыночной.

## Расчет закрытых охватов

Каждой частоте `n` сопоставляется закрытый охват `'Охват n+' − 'Охват (n+1)+'` и его TRP-вклад.

Максимальная частота расчета `N_max` определяется по TRP кампании:

| TRP | N_max |
|-----|-------|
| < 1200 | 25 |
| 1200–1700 | 40 |
| 1700–2200 | 50 |
| ≥ 2200 | 60 |

### Разбивка по зонам частот

**До `'Мин. частоты'` — неэффективные контакты:**

```
'TRP эфф.'   = 0
'TRP неэфф.' = ('Охват n+' − 'Охват (n+1)+') × n
```

**От `'Мин. частоты'` до `'Макс. частоты'` — эффективные контакты:**

```
'TRP эфф.'   = ('Охват n+' − 'Охват (n+1)+') × n
'TRP неэфф.' = 0
```

**От `'Макс. частоты' + 1` до `N_max − 1` — смешанная зона:**

```
'Закрытый охват' = 'Охват n+' − 'Охват (n+1)+'
'TRP эфф.'       = 'Закрытый охват' × 'Макс. частота'
'TRP неэфф.'     = 'Закрытый охват' × (n − 'Макс. частота')
```

**Строка `N_max+` — остаток TRP:**

```
'TRP строки'  = 'TRP кампании' − (сумма TRP уже учтённых строк)
'TRP эфф.'    = 'Охват N_max+' × 'Макс. частота'
'TRP неэфф.'  = 'TRP строки' − 'TRP эфф.'
```

## Итоговые значения

**Лист «Индекс»:**

| Поле | Описание |
|------|----------|
| Рекламодатель | Рекламодатель кампании |
| Кампания | Название кампании |
| TRP кампании | Суммарный TRP |
| Доля эфф. контактов кампании | TRP_eff / TRP кампании |
| Доля эфф. контактов рынка | Среднее по пулу |
| Оценка эффективных контактов | (Доля кампании / Доля рынка) × 100 |
| Границы эффективной частоты | Строка вида «Мин. частота–Макс. частота» |
| Количество кампаний в пуле | Размер выборки |

**Лист «Охваты»** — таблица с разбивкой TRP по частотам на эффективные и неэффективные контакты.

## Обновление пула

После расчета каждая кампания добавляется в пул (если её там ещё нет), чтобы будущие расчеты могли её использовать как референс.
