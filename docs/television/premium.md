# Позиционирование

Премиальное позиционирование — это размещение ролика на эффективных позициях в рекламном блоке (первая, вторая, предпоследняя и последняя).

## Входные данные

| Источник | Описание |
|----------|----------|
| Palomars | Основная выгрузка |
| Mediascope | TRP рынка по телеканалам и типу позиционирования за период кампании |

**Используемые столбцы:** Big TVR {ЦА}, Телеканал, Позиционирование

> Тематические и орбитальные каналы исключаются перед расчетом.

---

## Классификация позиционирования

Позиционирование приводится к двум категориям:

- **Премиальный** — все, кроме «Средний» (первая, вторая, предпоследняя и последняя позиции в блоке).
- **Средний** — строго «Средний».

---

## Расчет по кампании

По каждому телеканалу:

```
'Премиальный TRP камп.' = Σ Big TVR {ЦА}  (ролики с позиционированием 'Премиальный')
'TRP камп.'             = Σ Big TVR {ЦА}  (все ролики)

'Доля премиум камп.' = 'Премиальный TRP камп.' / 'TRP камп.'
```

---

## Расчет по рынку

Запрос к Mediascope за период кампании (только сетевое вещание, Big TV, SPOT, реальные выходы):

```
'Премиальный TRP рын.' = Σ TRP рынка  (позиционирование 'Премиальный')
'TRP рын.'             = Σ TRP рынка  (все позиционирования)

'Доля премиум рын.' = 'Премиальный TRP рын.' / 'TRP рын.'
```

---

## Индекс премиум позиционирования

```
'Индекс' = ('Доля премиум камп.' / 'Доля премиум рын.') × 100
```

При `'Доля премиум рын.' = 0` индекс принимается равным 0.

---

## Итоговые значения

По каждому телеканалу:

| Поле | Описание |
|------|----------|
| Телеканал | Название канала |
| Номер | Порядковый номер канала (для сортировки) |
| Премиальный TRP камп. | TRP премиальных размещений камп. |
| TRP камп. | Суммарный TRP камп. |
| Премиальный TRP рын. | TRP премиальных размещений рын. |
| TRP рын. | Суммарный TRP рын. |
| Доля премиум камп. | Премиальный TRP камп. / TRP камп. |
| Доля премиум рын. | Премиальный TRP рын. / TRP рын. |
| Индекс | (Доля камп. / Доля рын.) × 100 |

В конец таблицы добавляется строка «Итого» с агрегированными значениями и итоговым индексом.
