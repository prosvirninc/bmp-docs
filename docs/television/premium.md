# Позиционирование

Премиальное позиционирование — это размещение ролика на эффективных позициях в рекламном блоке (первая, вторая, предпоследняя и последняя).

## Входные данные

| Источник | Описание |
|----------|----------|
| Mediascope | Выгрузка по кампании: Big TVR {ЦА}, Телеканал, Позиционирование |
| Mediascope | TRP рынка по телеканалам и типу позиционирования за период кампании |

> Тематические и орбитальные каналы исключаются перед расчетом.

---

## Классификация позиционирования

Позиционирование приводится к двум категориям:

- **Премиальный** — всё, кроме «Средний» (первая, вторая, предпоследняя и последняя позиции в блоке).
- **Средний** — строго «Средний».

---

## Расчет по кампании

По каждому телеканалу:

```
'Премиальный TRP кампании' = Σ Big TVR {ЦА}  (ролики с позиционированием 'Премиальный')
'TRP кампании'             = Σ Big TVR {ЦА}  (все ролики)

'Доля премиум кампании' = 'Премиальный TRP кампании' / 'TRP кампании'
```

---

## Расчет по рынку

Запрос к Mediascope за период кампании (только сетевое вещание, Big TV, SPOT, реальные выходы):

```
'Премиальный TRP рынка' = Σ TRP рынка  (позиционирование 'Премиальный')
'TRP рынка'             = Σ TRP рынка  (все позиционирования)

'Доля премиум рынка' = 'Премиальный TRP рынка' / 'TRP рынка'
```

---

## Индекс премиум позиционирования

```
'Индекс' = ('Доля премиум кампании' / 'Доля премиум рынка') × 100
```

При `'Доля премиум рынка' = 0` индекс принимается равным 0.

---

## Итоговые значения

По каждому телеканалу:

| Поле | Описание |
|------|----------|
| Телеканал | Название канала |
| Номер | Порядковый номер канала (для сортировки) |
| Премиальный TRP кампании | TRP премиальных размещений кампании |
| TRP кампании | Суммарный TRP кампании |
| Премиальный TRP рынка | TRP премиальных размещений рынка |
| TRP рынка | Суммарный TRP рынка |
| Доля премиум кампании | Премиальный TRP кампании / TRP кампании |
| Доля премиум рынка | Премиальный TRP рынка / TRP рынка |
| Индекс | (Доля кампании / Доля рынка) × 100 |

В конец таблицы добавляется строка «Итого» с агрегированными значениями и итоговым индексом.
